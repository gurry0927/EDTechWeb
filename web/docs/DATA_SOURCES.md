# 資料來源與授權

## 地圖邊界資料

| 檔案 | 來源 | 授權 |
|------|------|------|
| `src/data/taiwan_counties.geojson` | 內政部國土測繪中心 (NLSC) 直轄市、縣(市)界線 | 政府資料開放授權條款第1版 |
