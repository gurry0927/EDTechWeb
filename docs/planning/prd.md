# PRD: 每週三分鐘：看地圖聊時事 (Weekly Map Story)

## 0. 專案背景與目標
當前教育環境面臨孩子注意力下降、過度依賴 AI 直接給答案，以及老師在面臨 108 課綱「素養題」命題壓力大的痛點。本專案透過「視覺化地圖互動」+「AI 啟發式引導」，旨在建立一個低維護、高價值的教育輔助系統。

## 1. 目標用戶 (User Persona)
- **教育工作者 (Teachers)**：需要具備時事感、視覺化、免安裝的備課教材。
- **家長 (Parents)**：希望小孩在 AI 時代保持思辨能力，尋找有深度且非娛樂導向的數位內容。
- **學生 (Students)**：厭倦背誦，需要視覺呈現來理解地理與社會概念。

## 2. 核心功能 (Core Features)

### A. 動態地圖標註系統 (Dynamic Map Marker System)
- **需求**：地圖組件必須與數據解耦，可透過 JSON 檔案動態載入特定經緯度的事件。
- **細項**：
    - 多種標籤類型（新聞事件、歷史景點、數據對比）。
    - 點擊標註展開「詳情側欄 (Side Panel)」。
    - 支援「時間軸」切換（例如：比較不同年份的鄉鎮邊界或數據）。

### B. AI 蘇格拉底引導組件 (AI Socratic Tutor Component)
- **需求**：嵌入一個 AI 互動框，其 Prompt 限制為「不給答案，只給線索與引導」。
- **功能**：根據當前地圖顯示的數據，反問學生問題。

### C. 自動化內容生成流程 (Auto-Content Pipeline)
- **需求**：利用 AI 抓取時事新聞，並將其格式化為地圖數據點。
- **自動化鏈**：時事抓取 -> LLM 提取地理位置與核心素養知識點 -> 生成 JSON 數據 -> 人工 10 分鐘審核。

## 3. 獲利模式思考 (Monetization)
- **Freemium**：基礎時事地圖免費推廣，獲取教師/家長信任。
- **Premium Subscription**：
    - 下載「配套素養練習卷」。
    - 完整 AI 引導式對話紀錄分析（讓家長看見孩子的思辨過程）。
    - 歷史存檔無限訪問。

## 4. 維護與規模化 (Maintenance & Scalability)
- **模組化**：`TaiwanMap` 組件應維持純粹，僅透過 Props 接收 `markers` 與 `config`。
- **輕量化**：全站 SSR (Next.js)，確保移動端載入速度，降低「注意力流失」。
- **社群力**：初期以「分享連結」到老師/家長社群獲取第一批種子用戶。

## 5. 成功指標 (Success Metrics)
- **短期**：第一個老師將連結分享到課堂上、收到第一則真實反饋。
- **長期**：核心用戶的平均停留時間（Focus Time）與 AI 對話的「深度」（問題產生的質與量）。
