# Vision Expanded: 跨學科時空學習系統 (The Multiverse of Learning)

這份文件記錄了我們「大聊特聊」後的創意火花，將作為未來開發深度垂直內容的指南。

## 0. 核心偉大設計：台灣時空層疊系統 (Spatial Intelligence Layer System)

這是我認為本專案最具備「素養爆炸力」的設計。透過技術手段，讓地圖不再是扁平的，而是具備「跨學科洞察」的動態載體。

- **技術架構**：建立一個「萬能圖層切換器」，讓用戶隨意疊加不同維度的數據，底層統一使用 **台灣地形高程圖**。
- **圖層範例**：
    - **圖層 A（歷史）**：史前文化分布 -> 清領三大圳 -> 日治糖鐵 -> 現代高鐵。
    - **圖層 B（地科）**：板塊邊界 ->斷層線 -> 火成岩/沉積岩分布 -> 國家公園範圍。
    - **圖層 C（社會）**：現代人口密度 -> 薪資水平 -> 各級政府預算分配 -> 產業園區分布。
- **素養引導（Killer Feature）範例**：
    - **案例 1：避開災害的智慧**
        - 當學生同時開啟「圖層 B（斷層/地質）」與「圖層 C（人口密度）」時，AI 蘇格拉底主動提問：**「你有發現為什麼大城市跟特定地質區域總是刻意保持距離嗎？」**
    - **案例 2：跨越時空的海陸變遷**
        - 當學生開啟「圖層 A（史前文化：圓山文化）」與「圖層 B（古地理地形：台北湖模擬）」時，AI 問：**「圓山文化遺址發現了大量的貝塚（ shells mounds），但現在圓山距離海岸線很遠，你覺得當時的人是去哪裡撿貝殼的？」**
        - 透過調整「海平面滑桿」，學生能親眼看見台北盆地在數千年前曾是個鹹淡水交會的溺谷（台北湖），瞬間理解「貝塚」存在的合理性。
    - 這種**跨學科的空間關聯性**，是傳統分科教學永遠無法觸及的深度。

## 1. 歷史：全球共時時段 (Timeline Synchronicity)
- **痛點**：台灣史、中國史、世界史切割，學生對同一個時間點的全球局勢缺乏空間感。
- **方案**：
    - **同步滑桿**：拖動時間軸（例如 1624 年），地圖同時標註：
        - 台灣：荷蘭人登陸大員。
        - 中國：明朝末年、閹黨亂政。
        - 世界：歐洲三十年戰爭中。
    - **價值**：讓學生建立「全球史觀」。

### 2. 數學：混亂數據的建模 (Real-world Modeling)
- **痛點**：數學是「紙上的完美」，學生不認同數學能解決真實世界的問題。
- **方案**：
    - **人口分布幾何化**：利用現有的人口密度數據，讓學生計算「行政中心的最短服務路徑」。
    - **非對稱圖形**：計算不規則海岸線的表面積，引導分形 (Fractal) 概念。
    - **價值**：從「負熵」中提取秩序，展示數學的權力。

### 3. 理化：具象化模擬層 (Science Visualization Layer)
針對學生最痛苦的記憶點，用地圖或動畫「畫出來」。

- **光學 (Optics)**：
    - **透鏡折射動態化**：在地圖的高程圖上模擬陽光照射，或是建立一個獨立的小工具，讓學生自由拉動透鏡，看見光線如何匯聚/發散。
- **化學 (Chemistry)**：
    - **互動式酸鹼地圖**：結合台灣各地的土壤酸鹼度數據，點擊區域顯示適合作物，並跳出相關的離子反應方程式。
    - **火焰顏色與沉澱表**：建立一個「元素地圖」，或者是一個「虛擬實驗室」，點擊不同的鹽類看沉澱反應，取代死背沉澱表。

## 下一步開發考量 (Roadmap)
